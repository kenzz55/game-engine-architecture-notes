게임 엔진 아키텍처를 학습하며 정리한 개인 학습 노트입니다.

## 목적

* 게임 엔진의 전체 구조와 계층적 설계를 이해
* 렌더링, 자원 관리, 애니메이션, 물리, 오디오, 네트워킹 등 주요 하위 시스템 정리
* 복잡한 아키텍처 구조를 Markdown 문서와 도식화 이미지로 정리
* 장기적으로 게임 클라이언트/엔진 프로그래밍 학습 기록으로 활용
